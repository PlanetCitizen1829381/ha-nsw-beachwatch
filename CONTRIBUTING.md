## How Can I Help?

### Reporting Bugs
* Check the [Issues](https://github.com/PlanetCitizen1829381/ha-nsw-beachwatch/issues) tab to see if the bug has already been reported.
* If not, open a new issue. Clearly describe the problem and include any relevant logs from Home Assistant.

### Suggesting Enhancements
* Open an issue to discuss your idea before writing any code. This ensures the feature fits the goals of the integration.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Ensure your code follows the existing style (no #comments, clean structure).
4. Update the `README.md` if you are adding new sensors or features.
5. Submit your Pull Request with a clear description of what you changed.

## Development Environment
This project uses the standard Home Assistant integration structure. To test your changes locally:
1. Copy the `custom_components/nsw_beachwatch` folder to your Home Assistant `config/custom_components/` directory.
2. Restart Home Assistant to load the changes.
